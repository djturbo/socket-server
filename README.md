

# Socket-Server

Instalar los módulos para Node
````
npm install
```

Generar El dist
```
tsc -w
````

Levantar el servidor:
```
nodemon dist/
o
node dist/
```

Si por ejemplo se borrar algún archivo y se quiere recuperar se ejecuta el siguiente comando:
```
git checkout -- .
````
